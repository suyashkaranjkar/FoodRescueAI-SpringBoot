# 🍲 FoodRescueAI: Smart Waste Management System

[![Java](https://img.shields.io/badge/Java-17%2B-orange?style=for-the-badge&logo=java)](https://www.oracle.com/java/)
[![Spring Boot](https://img.shields.io/badge/Spring%20Boot-3.x-brightgreen?style=for-the-badge&logo=springboot)](https://spring.io/projects/spring-boot)
[![MySQL](https://img.shields.io/badge/MySQL-Storage-blue?style=for-the-badge&logo=mysql)](https://www.mysql.com/)

**FoodRescueAI** is an intelligent food waste management platform that bridges the gap between surplus restaurant inventory and NGOs. Powered by a Java-based AI rule engine, the system automates real-time matching, logistics tracking, and expiry risk mitigation to ensure surplus food reaches those in need efficiently.

---

## ✨ Key Features

- 🏨 **Dual Dashboards:** Specialized interfaces for Hotels (Donors) and NGOs (Receivers).
- 🔐 **Secure Admin Portal:** Centralized control for managing users and monitoring rescues.
- ⚡ **AI Rule Engine:** Smart matching logic to decide the best receiver based on proximity and urgency.
- 🔍 **Live Search:** High-performance AJAX-based search for Admin and Hotel inventories.
- ✅ **Form Validation:** Robust backend and frontend validation to ensure data integrity.
- 🛡️ **Expiry Mitigation:** Real-time tracking of food "cooked time" vs "expiry date."
